# Bogie Flow — Standard Operating Procedures

## Monsoon injection (demo)
1. Open **Overview** → **Injection Controls**
2. Select segment (default S4) and trigger monsoon inject
3. Watch corridor risk matrix and agent logs for hydrology updates

## Anomaly response
1. Open **Analysis** for the affected segment
2. Review vibration signature and soil-rain correlation
3. Click **Authorize Deployment** to inject anomaly simulation and create maintenance tickets

## Maintenance workflow
1. Open **Maintenance** view (or **Network Logs** in footer)
2. Review P1/P2 tickets and agent log stream
3. Cross-reference segment state on **Station Map**

## Climate monitoring
1. Open **Climate** view for regional precipitation heatmap
2. Track asset longevity estimates and vibration shift table

See `docs/PROJECT.md` in the repository for full architecture documentation.
